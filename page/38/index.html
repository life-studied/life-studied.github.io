    2022-03-24 /
    • { 数据结构与算法lesson }

    图的遍历

    DFS

    递归（前中后序）或者栈实现

    缺点：不能保证找到的是最近的点

    优点：

    适用于搜索全部解或者必须走到最深处才能查找到解

    空间上不需要额外空间

    不需要保存搜索中的状态

    BFS

    队列（层序遍历）

    缺点：需要持续性防止重复操作

    优点：只要找到就是最近的点

    适用于搜索最近解或一个解
