使用命令行选项

1. 一些选项

1.1 禁用TCP/IP

不允许使用TCP/IP，而是用别的方式进行连接，详见01 Mysql服务器启动方式。

客户端使用-h连接失效：

ERROR 2003 (HY000): Can’t connect to MySQL server on ‘127.0.0.1’ (61)

2. 短形式与长形式

对于常用的选项可以使用短形式：

长形式 短形式 含义
–host -h 主机名
–user -u 用户名
–password -p 密码
–port -P 端口
–version -V 版本
