步骤

仅在github上使用

  • 在当前目录下创建版本仓库

  • 使用git add .

  • 使用git commit -m 提交信息

  • 在远端创建仓库，并复制URL

  • 如果没有创建personal access token，创建一个

  • git remote add

  • 使用git push --set-upstream <远程仓库名，一般设为origin> <远程分支名>

  • 使用git push

  • 如果需要personal access token，将上述的token输入给git
