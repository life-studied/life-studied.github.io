5.2.2 方法的参数传递机制

Java 里方法的参数传递方式只有一种：值传递

Java 对于引用类型的参数传递， 样采用的是值传递方式。

5.2.3形参数可变的方法

在定义方法时， 在最后一个形参的类型后增加三个点(…)，则表明该形参可 接受多个参数值，多个参数值被当成数组传入。

注意：个数可变的形参只能处于形参列表的最后。 也就是说，一个方法中最多只能有一个个数可变的形参。
