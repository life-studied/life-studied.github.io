概览

Core Framework Actors

  • Character/Pawn：游戏控制的对象

  • Controller：控制器，用于控制对象，同时也用于客户端向服务端发送请求的通道

  • Player State：存储着其它玩家需要知道的关于你的信息，例如你的得分和血量

  • Game State：存储游戏的综合信息，例如当前游戏阶段和剩余时间
